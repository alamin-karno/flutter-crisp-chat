# Support

## Get Help

If you're having trouble with the plugin, here's how to get help:

### Documentation

Start with the docs — most questions are answered here:

- [Common Issues](/troubleshooting/common_issues) — Build errors, notification problems, session issues
- [Platform-Specific](/troubleshooting/platform_specific) — Android and iOS specific problems
- [FAQ](/reference/faq) — Frequently asked questions

### GitHub Issues

For bugs, feature requests, or questions not covered in the docs:

1. Search [existing issues](https://github.com/alamin-karno/flutter-crisp-chat/issues) first
2. If your issue isn't listed, [open a new issue](https://github.com/alamin-karno/flutter-crisp-chat/issues/new)
3. Include your Flutter version, platform, plugin version, and steps to reproduce

### Crisp Help Center

For questions about the Crisp platform itself (not this Flutter plugin):

- [Crisp Help Center](https://help.crisp.chat/en/)
- [Crisp Android SDK docs](https://docs.crisp.chat/guides/chatbox-sdks/android-sdk/)
- [Crisp iOS SDK docs](https://docs.crisp.chat/guides/chatbox-sdks/ios-sdk/)
